# CSS-Hamburger-Menu-in-Mobile
A responsive navigation bar with a hamburger menu for Mobile Devices.
Responsive Navigation bar
Hamburger menu visible only on mobile views
Menu buttons hidden by default
uses the :focus pseudo-class to display the menu 
.hamburger-btn:focus + mobile-menu{
display:block;
}
In Desktop View Navigation links are displayed and hamburger menu icon is hidden
In Mobile view navigation links are hidden and hamburger menu visible

Displayed only Mobile screens using media queries
.hamburger-btn{
display:block;
}
Designed By 
HTML
CSS
Flexbox
Media Queries
Pseudo Class
Final output in Mobile View
Navigation menu are hidden
Hamburger menu icon becomes visible
Hamburger button hidden by default using display:none;
